💰 Budget App 
Simple budget tracker aanu… React + Tailwind use cheythu build cheytha. Money manage cheyyan oru chill tool.



🚀 Features
Add cheyyam income and expense (tea kada bill ninnum salary vare).

Summary section: Income / Expense / Balance — balance kandu manassu calm aakum.

Delete option: unwanted transaction remove cheyyam, like how we delete spam WhatsApp forwards.

UI: gradient + glassmorphism… kidu look.




🛠️ Tech Stack
React (vite setup, fast like KSRTC superfast 🚌).

TailwindCSS (utility classes, no tension).

PostCSS + Autoprefixer (background helpers).

git clone https://github.com/JilsJoseph3030/budget-app
cd budget-app
npm install
npm run dev
Open browser → http://localhost:5173  
App run cheyyum… money track cheyyuka, chill cheyyuka ✌️




🤓 Usage
Type description: “Tea kada”, “Petrol”, “Salary”.

Amount enter cheyyuka.

Select income or expense.

Add cheyyuka → summary update aakum.
